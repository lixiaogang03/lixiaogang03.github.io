## 对比

| 项目        | T153                | RK3506            |
| --------- | ------------------- | ----------------- |
| CPU       | 4× Cortex-A7 + E907 | 3× Cortex-A7 + M0 |
| 主频        | 约1.2GHz             | 约1.2~1.5GHz       |
| GPU       | 无3D GPU             | 无3D GPU           |
| NPU       | 无                   | 无                 |
| 视频编解码     | 较弱                  | 很弱                |
| 工业总线      | CAN-FD、LocalBus丰富   | CAN丰富             |
| 网口        | 最多3个GMAC            | 通常2个              |
| Linux支持   | Tina Linux          | Linux             |
| Android支持 | 几乎无                 | 几乎无               |
| 定位        | 工业控制器               | HMI控制器            |
