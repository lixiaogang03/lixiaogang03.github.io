## 安装 ROS2 Humble

**设置 locale**

```bash

sudo apt update
sudo apt install locales

sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8

export LANG=en_US.UTF-8

```

**添加源**

```bash

sudo apt install software-properties-common
sudo add-apt-repository universe

```

**添加 ROS key**

```bash

sudo apt update && sudo apt install curl -y

sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg

```

**添加仓库**

```bash

echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu \
$(. /etc/os-release && echo $UBUNTU_CODENAME) main" | \
sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

```

**安装**

```bash

sudo apt update

sudo apt install ros-humble-desktop -y

```

**环境变量**

```bash

echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc

source ~/.bashrc

```

**测试 ROS2**

```md

ros2 run demo_nodes_cpp talker

另开终端：

ros2 run demo_nodes_py listener

能通信说明 OK。

```

## 安装 TurtleBot3

这是 ROS 世界 Hello World。

[TurtleBot3](https://emanual.robotis.com/docs/en/platform/turtlebot3/overview/?utm_source=chatgpt.com)

**安装**

```md

sudo apt install ros-humble-turtlebot3* -y

```

**设置型号**

```md

echo 'export TURTLEBOT3_MODEL=burger' >> ~/.bashrc

source ~/.bashrc

```

**清理环境**

```md

执行：

pkill -9 gzserver
pkill -9 gzclient
pkill -9 ros2

然后：

ps -ef | grep gz

确保没有：

gzserver

```

**启动仿真**

```md

ros2 launch turtlebot3_gazebo empty_world.launch.py

```

## 新老架构仿真

| 老              | 新               |
| -------------- | --------------- |
| Gazebo Classic | Gazebo Harmonic |
| gzserver       | gz sim          |
| gzclient       | 内置GUI           |
| OGRE1          | OGRE2           |
| ROS插件          | ros_gz_bridge   |
| 容易崩            | 稳很多             |

## 切换到新架构

**移除旧架构**

```bash

sudo apt remove gazebo* libgazebo* -y

```

**安装 Harmonic**

```bash

sudo apt update
sudo apt install gz-harmonic -y

```

**测试**

```bash

gz sim

```

![gazebo_harmonic](/images/2026/0514/gazebo_harmonic.png)

**安装 ROS2 Bridge**

```bash

sudo apt install ros-humble-ros-gzharmonic -y

```

**测试 ROS 通信**

```md

启动：

gz sim shapes.sdf

另一个终端：

ros2 topic list

```
