## Nvidia 开源家族

| 模型家族                    | 定位                | 主要应用                |
| ----------------------- | ----------------- | ------------------- |
| **Nemotron**            | 大语言模型 / Agent     | 编程、推理、企业 AI         |
| **Cosmos**              | 世界模型（World Model） | 机器人、无人车、Physical AI |
| **GR00T**               | 机器人基础模型           | 人形机器人               |
| **Alpamayo**            | 自动驾驶模型            | 自动驾驶感知与规划           |
| **BioNeMo**             | 生物 AI             | 药物研发、蛋白质设计          |
| **Eagle / VILA / NVLM** | 多模态视觉模型           | OCR、图像、视频理解         |

## Meta 开源家族

| 家族              | 定位      | 应用           |
| --------------- | ------- | ------------ |
| **Llama**       | 通用 LLM  | 对话、推理、Agent  |
| **Code Llama**  | 编程模型    | 代码生成、补全      |
| **Llama Guard** | 安全模型    | 内容审核、安全防护    |
| **Llama Stack** | 开发框架    | Agent、RAG、部署 |
| **SAM / SAM 2** | 图像与视频分割 | 视觉、机器人、医学    |
| **Emu**         | 多模态生成   | 图文生成、图像理解    |
| **AudioCraft**  | 音频生成    | 音乐、环境声音      |
| **ImageBind**   | 多模态表示学习 | 跨模态理解、机器人    |
