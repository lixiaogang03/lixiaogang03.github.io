## 环形麦克风

| 对比项          | 环形麦克风      | 双 pin 麦克风    |
| ------------ | ---------- | ------------ |
| **灵敏度**      | 高          | 中等           |
| **信噪比（SNR）** | 往往更高       | 普通           |
| **拾音角度**     | 360° 全向    | 全向或单向        |
| **抗风噪**      | 更好         | 较一般          |
| **远距拾音**     | 更好（>3–5 米） | 较弱（1 米以内最清晰） |
| **成本**       | 高          | 低            |
