## 环形麦克风

| 对比项          | 环形麦克风          | 双 PIN 麦克风     | 适用场景/说明           |
| ------------ | -------------- | ------------- | ----------------- |
| **灵敏度**      | 高              | 中等            | 灵敏度高意味着远场声音也能采集到  |
| **信噪比（SNR）** | 往往更高           | 普通            | SNR 高 → 语音更清晰，噪声低 |
| **拾音角度**     | 360° 全向        | 全向或单向         | 全向拾音可覆盖多人或会议环境    |
| **抗风噪**      | 更好             | 较一般           | 适合室外或风噪环境使用       |
| **远距拾音**     | 更好（>3–5 米）     | 较弱（1 米以内最清晰）  | 适合远场语音识别或会议       |
| **成本**       | 高              | 低             | 成本高一般用于专业设备       |
| **推荐用途**     | 会议系统、智能音箱、远场语音 | 手机、平板、普通录音/唤醒 | 可快速选择场景应用         |

**4 通道差分 ADC 接口麦克风阵列**

1. 4 通道：阵列里有 4 个独立的麦克风，每个麦克风都能单独收音。
2. 差分麦克风：每个麦克风是差分输出，对噪声（尤其是电磁干扰）抑制能力强。
3. 阵列：麦克风按一定几何方式排列，可以通过“相位差”分析声音方向，实现定向拾音或声源定位。

**降噪和回声消除**

| 功能   | 硬件依赖           | 算法依赖           | 原理                    |
| ---- | -------------- | -------------- | --------------------- |
| 降噪   | 高（差分麦克风、阵列方向性） | 可选（算法可进一步增强）   | 物理上弱化噪声或通过方向性拾音降低环境噪声 |
| 回声消除 | 低              | 必须（预测扬声器回声并减去） | 软件算法减去已知的扬声器声波在麦克风的叠加 |

```markdown

           前方人声
               ↑
               │
          ┌─────────┐
          │   M1    │
          │         │
M4 ───────┤         ├────── M2
          │         │
          │   M3    │
          └─────────┘
               │
               ↓
         扬声器回声

说明：
- M1~M4：4 个麦克风
- ↑ 前方人声：目标声音，阵列增强（Beamforming）
- ↓ 扬声器回声：从扬声器播放到麦克风的声音，算法预测并消除
- ↘ ↙ ↗ ↖：房间环境噪声，阵列衰减（降噪）

```
