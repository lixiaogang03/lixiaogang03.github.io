## 香港-香城两日

### 4月7日行程

* 7点在酒店早餐
* 8点半去深圳北站电信营业厅更换手机卡，以便开通香港漫游套餐
* 和朋友乘坐高铁到达香港西九龙站，出关
* 乘坐香港地铁到达中环
* 步行至兰芳园吃午餐，价格几乎是深圳的三倍
* 步行至太平山缆车总站

* 朋友游览山顶后返回深圳
* 独自在山顶咖啡馆喝咖啡，吃麦当劳
* 乘坐缆车回到中环
* 从中环步行至维多利亚港
* 沿着维多利亚港港岛滨海长廊散步，休息，阅览香港纸质地图
* 晚上7点乘坐天星小轮前往九龙半岛
* 沿着九龙半岛游览维多利亚港夜景
* 晚上九点步行至龙堡国际酒店，在酒店门口吃潮州小吃，猪肚鸡面，60港元
* 回酒店休息，购买英标手机充电插头，酒店价格500港币，比清明节时候便宜，是深圳光明酒店价格的两倍

## 深圳感受

1. 深圳是丘陵地带，平地少，免费的丘陵公园很多
2. 深圳的城市面貌和基础设施都极好
3. 福田和南山是绝对的中心，当然房价也高 10万/平米
4. 朋友居住的光明区房价想对低很多，4万/平米
5. 深圳北站人很多，早晚高峰很拥挤

## 香港感受

1. 去香港的手续跟去另一个国家似乎差不多，港澳通行证类似护照，签注类似签证，只是名字不同
2. 乘坐高铁去香港非常方便，比从深圳坐地铁过关要方便很多
3. 香港是自由港，外国游客和外籍员工(主要在中环)都很多，东南亚的佣工也多，主要从事低端服务行业
4. 香港是购物天堂，到处都是店铺
5. 香港没有外卖，没有两轮电动车，摩托车也少
6. 香港街道非常干净，乱扔垃圾罚款极重
7. 香港地铁不允许吃东西
8. 楼非常高，但街道很窄，居住条件不太好，但是秩序很好
9. 小店铺普遍说粤语，高档的店铺可以普通话交流，商务精英普遍英语交流
10. 吃住的费用是深圳的两倍，节假日酒店费用会更贵
11. 香港双层巴士发达，是除地铁外的主要交通方式
12. 维多利亚港和上海外滩景色类似，但是对游客更友好，休息的地方多
13. 香港公共卫生间极少，因为土地稀缺，只能去大型商场找公共卫生间
14. 便利店也很贵，7-Eleven店铺很多，一瓶矿泉水需要10港币
15. 酒店充电插头是英国标准，需要购买转换插头
16. 酒店空间比深圳小的多，不提供一次性牙刷，灯光也比较暗，不提供免费矿泉水，酒店窗户也都不可打开
17. 公共场所也非常安静，很少人大声喧哗，路人走的很快，绿灯时有铃声催促你快速过马路
18. 九龙半岛上有很多老楼，用竹制脚手架给楼外观翻新
19. 支付方式：微信和支付宝已经很普及了，苹果支付绑定八达通也很方便
20. 餐厅拼桌吃饭是常态，吃完就走，很少久坐
21. 深圳博物馆在讲城市是怎么被打造出来的，香港历史博物馆在说生活是怎么演变的
22. 香港其实很大，但是山多，人主要集中在九龙区和香港岛
23. 香港岛太平山是必去景点，可以俯瞰整个维多利亚港，山上名人别墅众多

**总结**

香港是一个在“极端空间约束 + 成熟规则体系”下运行的高效率城市，而内地城市是在“更大空间 + 快速发展”中不断优化体验的系统

## 跨境支付通

```md

【用户操作】
内地App（如银行App/微信/支付宝）
   ↓
【银行层（内地）】
付款银行（冻结余额 + 风控审核）
   ↓
【清算层（内地）】
IBPS（小额跨行清算系统）
   ↓
【跨境桥】
跨境支付通 / 跨境清算接口（CIPS / 银联 / 银行直连）
   ↓
【外汇层】
人民币 → 港币（银行或做市商实时换汇）
   ↓
【香港清算层】
FPS（转数快）
   ↓
【银行层（香港）】
收款银行入账
   ↓
【用户到账】
香港账户余额增加


```

### 案例

```md

假设：

你从内地转 1000 RMB 到香港

实际发生：
① 内地银行  --- 扣你 1000 RMB
② 香港银行  --- 给对方 +1100 HKD
③ 中间关键（你看不到的）--- 两家银行之间有：往来账户（Nostro/Vostro）或通过清算系统结算

银行通常是：一天或者一段儿时间做一次清算

```

###  SWIFT 对比 CIPS

| 项目    | SWIFT | CIPS   |
| ----- | ----- | ------ |
| 是否转钱  | ❌ 否   | ✅ 是    |
| 是否清算  | ❌ 否   | ✅ 是    |
| 是否系统级 | 消息层   | 金融基础设施 |
| 是否独立  | 各币种通用 | 人民币专用  |

**SWIFT** 是：

* 注册地：比利时（欧洲）
* 法律监管：欧盟 + 比利时央行
* 成员制组织：全球银行共同拥有

一个“全球银行联盟 + 欧洲法律管辖”的中立通信网络

### 双边结算的问题

**流动性不足（核心痛点）**

```md

🌍 美元体系：
1. 全球通用
2. 随时可换
3. 深度市场（liquidity deep）

👉 可以理解为：

“全球统一的现金池”

🪙 双边结算：
人民币 ⇄ 卢布
人民币 ⇄ 里亚尔

问题是：

1. 卢布换人民币要找对手方
2. 市场深度小
3. 大额交易容易“滑点”

📌 结果：

👉 “钱不是不能换，而是不好换 + 换不好价”

```

**汇率机制弱**

```md

美元体系：
1. 有全球外汇市场（FX）
2. 定价连续
3. 有大量套利者

双边结算：
1. 交易量小
2. 市场不连续
3. 参考价不稳定

📌 结果：

汇率容易“人为锚定”
或者靠协议价（非市场价）

👉 直接后果：

企业承担“隐性汇率成本”

```

**信用体系弱**

```md

美元体系优势：
1. SWIFT + 国际银行信用链
2. 有统一清算规则
3. 违约成本高


双边体系问题：
1. 依赖双边银行协议
2. 风险集中在国家层面
3. 中间清算透明度低

📌 结果：

👉 “信用不是市场化的，而是政治/协议化的”

```

**资金“卡住”的风险**

```md

这是企业最真实痛点：

出口 → 收到卢布
但：无法快速换回人民币

原因：

1. 外汇储备不平衡
2. 对方货币需求不足
3. 银行配额限制

👉 结果叫：

“settlement imbalance（结算不平衡）”

```

**贸易结构扭曲**

```md

双边结算会带来一个副作用：

👉 “贸易必须对称”

比如：

中国卖工业品给俄罗斯
但俄罗斯没等值商品回流

📌 结果：

顺差积累成“对方货币资产”
资金被困在体系里

```

**为什么人民币国际化最大瓶颈不是政策而是“资产池”**

外国人拿到人民币以后，有没有地方长期放、赚收益、还能随时进出

| 维度            | 🇺🇸 美元体系         | 🇨🇳 人民币体系          |
| ------------- | ----------------- | ------------------- |
| **安全资产（核心锚）** | 美债（全球最大、安全+流动性极强） | 中国国债（规模大，但全球参与度较低）  |
| **信用体系**      | 全球统一信用定价（评级+市场化）  | 多层信用结构，市场化程度仍在提升    |
| **股票市场**      | 全球最大（纳斯达克+NYSE）   | A股规模大，但外资占比低、波动结构不同 |
| **公司债市场**     | 深度极高（美元公司债全球配置）   | 境内为主，离岸市场规模较小       |
| **衍生品市场（关键）** | 超级发达（利率/汇率/信用衍生品） | 在发展中，工具种类与流动性不足     |
| **外汇对冲能力**    | 完整（期货、掉期、期权全球流动）  | 相对有限，深度不够           |
| **全球接纳度**     | 全球储备货币（占主导）       | 区域化使用（贸易货币属性更强）     |
| **资本流动性**     | 高度自由流动            | 有管理的开放（存在边界）        |
