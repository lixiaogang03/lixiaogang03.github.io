## 现象

更换电池后开了三个月，七天前开还正常，车库停了七天后，再启动时出现的故障，仪表盘提示自动驻车故障，发动机起不来，但是车机起来了

更换电池之前静态停放两周也没问题

## AI 分析原因

👉 车库停 7 天 → 电池电压缓慢下滑 → BCM/EPB 被判定为不安全
👉 进入“允许车机上电，但禁止启动车辆”的保护模式

1️⃣ 星瑞是“高静态耗电车型”

即使熄火锁车，仍在工作的模块包括：

* BCM
* 防盗系统
* 车机待机
* 通讯模块（T-Box）

👉 每天 30–70 mA 并不少见

## 可能原因1-更换电池后未重置数据

概率较小

## 可能原因2-异常静态耗电

地下车库长期静置 → 异常静态耗电 → 电压跌破 BCM 启动阈值 → 系统强保护

| 静态电流         | 结论      |
| ------------ | ------- |
| **≤ 40 mA**  | 正常      |
| 50–80 mA     | 偏高      |
| **≥ 100 mA** | 一定有模块没睡 |

## 为什么这次只停 7 天就出问题？

1️⃣ 新电池 + 内阻低 + SOC 模型敏感

* 新电池电压下降快，IBS 判断更严格
* 一旦静态耗电略高 → 直接触发保护

2️⃣ 地下车库环境 + 模块没睡

* T-Box 或车机 MCU 长时间唤醒
* 造成异常静态耗电 100–300 mA

3️⃣ 偶发因素

* 停车期间系统偶尔尝试远程通信 / 定位
* 微小异常被累积，导致 7 天就“越界”

所以出现“以前两周没事，现在 7 天就锁死”的现象，完全是电池管理逻辑 + 模块静态耗电叠加导致，而不是车坏或电池问题。

## 原因总结

| 现象           | 解释      |
| ------------ | ------- |
| 高速 400 km 正常 | 行驶充电没问题 |
| 地下车库停 7 天    | 静态耗电放大  |
| 车机能亮         | 低风险供电层  |
| 钥匙失效         | BCM 被保护 |
| Auto Hold 故障 | EPB 被锁  |
| 搭电立刻恢复       | 低压问题实锤  |

## 排查清单

| 步骤                        | 操作内容                   | 位置 / 模块       | 判定标准                             | 备注               |
| ------------------------- | ---------------------- | ------------- | -------------------------------- | ---------------- |
| **准备工作**                  | 准备万用表、OBD诊断电脑、安全工具     | 车辆电瓶及诊断接口     | 无                                | 等车锁车30分钟以上，让车辆休眠 |
| **测整车静态电流**               | 串联万用表测电瓶负极电流           | 电瓶负极          | ≤40 mA 正常；50–80 mA 偏高；≥100 mA 异常 | 记录数值，为后续排查提供参考   |
| **模块排查：T-Box**            | 拔模块保险或断开连接             | 副驾驶下方/中控附近    | 电流明显下降 → T-Box耗电                 | 地下车库信号弱时易唤醒      |
| **模块排查：中控 MCU / 车机主机**    | 断开车机电源                 | 中控屏幕后/仪表台内    | 电流下降 → 车机休眠异常                    | 可能受系统更新或软件影响     |
| **模块排查：BCM / EPB / PEPS** | 拆保险或断开模块               | 仪表台下方/车门      | 电流下降 → 模块异常                      | PEPS天线靠近钥匙易唤醒    |
| **外接设备排查**                | 拔掉 USB、行车记录仪、OBD线      | USB接口/电源接口    | 电流下降 → 外接设备耗电                    | 检查改装设备或临时接线      |
| **线路与电路检查**               | 检查电瓶端子、线束、保险丝、继电器      | 电瓶端子/主线束/保险丝盒 | 电流异常 → 存在短路或接触不良                 | 排查机械故障或漏电        |
| **软件与系统检查**               | 读取 SOC、IBS 数据，查看模块休眠状态 | OBD / 诊断电脑    | 模块未休眠或电流异常 → 软件/策略问题             | 升级软件或刷新休眠策略      |
| **记录与判定**                 | 每次操作记录电流变化             | 全车            | 拆掉某模块电流下降明显 → 确定耗电源头             | 提供给维修或厂家根治       |
| **临时防护措施**                | 定期启动/断负极/维护充电器         | 电瓶/车辆         | 防止低压保护触发                         | 短期停放使用，防止复发      |
