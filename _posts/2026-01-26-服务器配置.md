**CPU配置**

| 属性      | 值                                 | 说明                         |
| ------- | --------------------------------- | -------------------------- |
| 架构      | x86_64                            | 支持 64 位操作系统                |
| 核心数     | 32 个线程，16 核 × 2 个座                | 32 线程，双路物理 CPU，每颗 CPU 16 核 |
| 每核线程数   | 2                                 | 支持超线程（Hyper-Threading）     |
| 频率      | 1 GHz 基础，4 GHz 最大                 | 可动态调节，节能与性能兼顾              |
| CPU 型号  | Montage Jintide C4215R            | 双路服务器级 CPU                 |
| 缓存      | L1d/i 512 KiB，L2 16 MiB，L3 22 MiB | 较大缓存，适合高并发计算               |
| NUMA 节点 | 2                                 | 内存访问分布在两个物理 CPU            |
| 虚拟化     | VT-x                              | 支持硬件虚拟化                    |


**双路CPU**

* 每颗 CPU：16 核 × 2 线程（超线程） = 32 线程
* 双路：两颗 CPU → 总共 32 核 × 2线程/核 = 64 线程

## 通过 iBMC 地址访问服务器

iBMC 并不是完整的通用操作系统，它是一个嵌入式管理系统/固件，运行在服务器主板的独立管理芯片上（BMC 芯片），与主机操作系统完全独立

**通过iBMC地址访问服务器**

开机过程中会显示iBMC地址, 使用账号密码登陆即可
