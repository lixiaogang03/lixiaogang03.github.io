## 存储芯片分类

```markdown

存储芯片
├── 易失性存储（断电丢数据）
│   └── DRAM
└── 非易失性存储（断电不丢）
    ├── NAND Flash
    └── NOR Flash

```

## DRAM

* DDR3 / DDR4 / DDR5（PC / 服务器）
* LPDDR4 / LPDDR5 / LPDDR5X（手机 / SoC）
* HBM（AI / GPU）

## 贴片厂

**主流做法**

```markdown

你（芯片设计方 / 板卡方）
   ↓ 采购
原厂 / 授权代理 / 模组厂
   ↓ 发料
贴片厂（SMT）
   ↓
贴片 + 回流

```

**贴片厂代采（适合成熟、小批量）**

风险点（一定要知道）

型号 OK，批次不可控，可能是：

* 翻新料
* 拆机料
* 非官方流通渠道

**贴片数量和价格的关系**

| 批量              | SMT 基础贴片费  | BGA / 精密工艺费 | 功能测试 / 烧录费 | 总估算成本（不含料）  | 备注           |
| --------------- | ---------- | ----------- | ---------- | ----------- | ------------ |
| 少量（10–50 块）     | 60–120 元/块 | 30–70 元/块   | 20–50 元/块  | 100–240 元/块 | 样机阶段，人工调机成本高 |
| 中等批量（200–500 块） | 30–60 元/块  | 15–40 元/块   | 10–25 元/块  | 55–125 元/块  | 重复贴片效率高，成本下降 |
| 大量（1000 块以上）    | 20–40 元/块  | 10–25 元/块   | 5–15 元/块   | 35–80 元/块   | 量产最优，贴片厂效率高  |

**代理**

| 级别   | 适合用途       | 注意事项           |
| ---- | ---------- | -------------- |
| 一级代理 | 量产 / 核心件   | 起订量较高，价格稳定，可锁料 |
| 二级代理 | 样机 / 小批量   | 灵活，但批次/库存有限    |
| 三极代理 | EVT / 临时补货 | 高风险，不可依赖长期供货   |
