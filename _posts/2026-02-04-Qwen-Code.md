## Qwen Code VS Claude Code  

| 项目     | Qwen Code CLI      | Gemini CLI                | Claude Code                |
| ------ | ------------------ | ------------------------- | -------------------------- |
| 定位     | 开源 AI 编程终端工具       | 开源 AI 编程终端工具（Google 出品）   | 商业 AI 编程助手                 |
| 底层模型   | 🧠 Qwen3‑Coder（阿里） | 🧠 Gemini 2.5 Pro（Google） | 🧠 Claude 系列（Anthropic）    |
| 是否开源   | ✅ 是（Apache 2.0）    | ✅ 是（Apache 2.0）           | ❌ 不是开源                     |
| 主要输出形式 | CLI 命令 + 脚本式交互     | CLI + 多模态支持               | 多模式助手（终端 + VSCode + Cloud） |

## Claude Code VS Open Code

OpenCode 的运营主体并不是一个大公司（像 OpenAI 或 Google 那样的法人企业），而是一个开源社区项目，主要由“OpenCode 团队/组织”维护和推动

| 维度    | Cloud Code | OpenCode  |
| ----- | ---------- | --------- |
| 模型    | 固定官方模型     | 任意模型      |
| 本地模型  | ❌          | ✅         |
| 成本    | 高          | 可低        |
| 隐私    | 需上传代码      | 可本地运行     |
| 大项目能力 | ⭐更强        | 依赖模型      |
| 灵活性   | 低          | ⭐极高       |
| 适合人群  | 企业/团队      | 个人开发者/研究者 |

## 模型能力对比

| 能力            | Qwen Code CLI                 | Gemini CLI                     | Claude Code                   |
| ------------- | ----------------------------- | ------------------------------ | ----------------------------- |
| **代码生成能力**    | 强（同样注重代码内容）                   | 很强                             | 很强且偏稳定、逻辑深度强                  |
| **上下文处理能力**   | 支持大上下文但略逊于 Gemini             | 💪 巨大上下文（1M tokens）            | 有上下文支持，但通常较 Gemini 少          |
| **项目理解/任务拆分** | 能执行复杂任务，但成熟度略弱                | 通用命令强，但具体算法略常规                 | 专业设计支持 Plan/Project 工作流       |
| **工具调用&扩展**   | 支持插件扩展（兼容 Gemini & Claude 扩展） | 支持扩展 & Google Search Grounding | 支持 Model Context Protocol 但闭源 |

## 成本

| 方面          | Qwen Code CLI | Gemini CLI        | Claude Code       |
| ----------- | ------------- | ----------------- | ----------------- |
| **免费使用门槛**  | ✔ 非常友好        | ✔ 免费强力            | ❌ 需要订阅 + token 计费 |
| **付费/订阅**   | ❌ 无商业必要       | ❌ 即使付费也主要依赖模型 API | ✔ 多档订阅计划          |
| **本地/私有部署** | ✔ 支持          | ✔ 支持              | ⚠️ 主要官方托管         |

## Qwen-Code CLI

Qwen3-Coder-Plus（强大的“大脑”）：这是理解任务、生成方案的智能核心，经过海量数据训练，能洞察复杂任务。
Qwen-Code CLI 工具（易用的“驾驶舱”）：这是一个安装简单的命令行工具，让你能轻松与“大脑”对话，指挥它完成任务。无论你是开发者想提升效率，还是办公人员想自动化处理文档、表格，它都能帮上忙。
