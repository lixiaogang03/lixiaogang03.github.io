## 常见发布方式

| 方式              | 适用场景        | 公开访问 | 难度   |
| --------------- | ----------- | ---- | ---- |
| Maven Central   | 开源库、公共 SDK  | 是    | ★★★★ |
| GitHub Packages | 团队内部、私有 SDK | 可私有  | ★★   |
| JitPack         | 开源项目快速发布    | 是    | ★    |
| 阿里云 Maven 仓库    | 国内团队        | 可私有  | ★★   |
| Nexus 私服        | 企业内部        | 私有   | ★★★  |
| Artifactory     | 大型企业        | 私有   | ★★★★ |
| 本地 AAR 文件       | 客户定制项目      | 否    | ★    |

## 个人app开发创业模式

```md

Android App
      ↓
Google Play 发布
      ↓
腾讯云服务器
      ↓
MySQL / Redis / COS

```

### 准备阶段

```md

Gmail账号（长期使用）
      ↓
国际信用卡（Visa/MasterCard）
      ↓
个人身份信息（身份证/护照）
      ↓
可访问Google服务的网络环境

```

### 注册开发者账号

```md

进入 Play Console
https://play.google.com/console/
      ↓
登录 Google 账号
      ↓
选择
👉 Personal Account（个人开发者）
      ↓
填写开发者信息
（姓名 / 地址 / 邮箱 / 电话）
      ↓
支付 $25 注册费
      ↓
账号激活

```

### 身份与账号验证

```md

身份验证（身份证/护照）
      ↓
手机号验证
      ↓
开发者资料审核
      ↓
（部分账号需要）
Closed Testing（14天测试）
      ↓
通过后解锁正式发布权限

```

### App 开发准备

```md

Android Studio 开发 App
      ↓
生成 AAB 包（不是 APK）
      ↓
配置：
- package name
- version code
- signing key

```

### 上架 Google Play

```md

⑤ 上架 Google Play
────────────────────────
进入 Play Console
      ↓
Create App（创建应用）
      ↓
填写商店信息：
- App名称
- 描述
- 图标
- 截图
- 分类
      ↓
上传 AAB
      ↓
填写：
- 隐私政策URL
- 数据安全表
- 内容评级
      ↓
提交审核


⑥ 审核与发布
────────────────────────
Google审核（1～7天）
      ↓
通过 → 上架
      ↓
全球用户可下载

```

### 收款流程

```md

⑦ 收款流程（关键）
────────────────────────
设置 Payments Profile
      ↓
绑定银行账户
（推荐：HSBC HK / 中国银行 / 招商银行）
      ↓
填写 SWIFT / IBAN / 地址
      ↓
收入结算（美元）
      ↓
每月自动打款

```

### 长期运营

```md

⑧ 运营阶段
────────────────────────
数据分析（Play Console）
      ↓
用户增长
      ↓
优化留存
      ↓
订阅 / 内购变现（Google Play Billing）

```

## 廊坊程序员案例启示

| 风险维度 | ❌ 高风险/违法违规路径（该案教训） |  合规/合法避坑路径 |
| :--- | :--- | :--- |
| **网络信道** | 个人私自使用未经国家批准的 **VPN/翻墙工具** 连接海外服务器工作或接单。 | 申请使用中国电信、中国联通等运营商提供的**正规跨境数据专线**，或使用合规登记的跨境加速服务。 |
| **业务主体** | 以**个人居民身份证**名义长期、高频直接对接境外公司承接离岸外包业务。 | 在国内注册**个体工商户或初创公司**。以企业主体对外签约，走正规的“服务贸易出口”流程。 |
| **资金结汇** | 境外公司直接将美元电汇至国内**个人银行卡**，或使用 Wise 等工具无申报结汇。 | 1. 货款打入**香港银行账户（如汇丰香港）**留存外币。<br>2. 通过**万里汇、空中云汇**等持牌跨境支付机构进行外贸订单合规申报，无限额结汇。 |
| **税务合规** | 认为赚取的是境外资金，国内无法联网查到，因而**不申报、不缴纳**任何税款。 | 依法进行个人所得税（**境外劳务报酬**）申报。若成立企业，可走正规出口服务贸易流程，甚至可享受国家**免税或退税**政策。 |
| **知识产权** | 个人私下接单，没有正规合同，软件著作权和代码归属权界定模糊。 | 签署正式的**离岸服务外包合同**，并在国内办理**软件著作权登记**或技术出口合同登记，作为合规收入的铁证。 |
